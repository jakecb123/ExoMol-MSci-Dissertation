# N2 Line List Submission

My dissertation is titled: "ExoMol line lists – the rovibrational quadrupole spectrum of the N2
ground-state".

This repository contains the scripts, input data, and supporting files used to generate the ExoMol linelists, to compare them with the existing HITRAN linelists, and all subsequent analysis.

- `BOB_correction.ipynb`, `additional_plots.ipynb`, and `linelist_analyser.ipynb` are the main analysis notebooks.
- `Duo/`, `ExoCross/`, `HITRAN/`, and `BOB_correction_data/` contain the data required by the notebooks.
- `ExoMol_format/` contains zipped ExoMol-style packages for each isotopologue.
- `Documents/` is for the dissertation writeup, outline, and presentations.
- `script_outputs/` is for generated notebook outputs.
